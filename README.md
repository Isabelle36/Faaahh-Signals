# Faaahhh Signals 🔊

Get hit with the **FAAAAH** the moment you type a wrong command.

## Features
- 🔊 Plays sound on every terminal error (non-zero exit code)
- 🎵 11 built-in meme sounds — Faahh, John Cena, Anime Girl, Gawk Gawk and more
- ⚡ Instant playback with zero delay
- 🎛️ Pick your sound via `Ctrl+Shift+P` → **Faaahhh: Pick Sound**
- 🔇 Enable/disable anytime in Settings

## Usage
Just install and break something. You'll hear it. 😂
