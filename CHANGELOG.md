All notable changes to Faaahhh Signals will be documented here.
[0.0.1] - 2026-03-01
Added

🔊 Plays error sound when a terminal command exits with a non-zero code
🎵 Sound pack with 11 built-in sounds including the classic Faaahh, John Cena, Anime Girl, Gawk Gawk, and more
⚙️ Sound picker dropdown in VS Code Settings — search "Faaahhh" to switch sounds
🔇 Enable/disable toggle in settings
🧪 Test sound command via Ctrl+Shift+P → "Faaahhh: Test Sound"
⚡ Persistent PowerShell process for near-instant playback with no delay
🛡️ Ignores Ctrl+C cancellations (exit code 130), agent terminals, and package install commands
🖥️ VS Code Task failure detection
